What is the Projects Gizmo?

The PROJECTS GIZMO allows you to create a repository of work and information relating to some activity you have accomplished.  This Gizmo is found using the "Gizmos" tab on the toolbar.

Quick Instructions:

  • Click on the GIZMO tab
  • Click on the Projects Gizmo
  • Click in your portfolio where you want the content to go
  • Give the project a title and description
  • Click on the checkmark to save the project and open the tutorial
  • View the tutorial information then jump in and create your project

 

Using the Projects Gizmo

From the GIZMOS tab, select the Projects Gizmo and click into the page were you want the gizmo to be placed.  The projects gizmo will open up and you will be prompted to give the project a title and description.  Once that is done, the 'Create' checkmark will be activated and you can create the project.

 

 

When you create your first project, a short tour of the projects gizmo will pop up.  It will introduce you to the different areas of the project.  It only appears the first time you create a project.

 

Building the Project

The first thing you are prompted to do is add an image to represent your project.  Simply click on the 'Change Image' banner on the image that is automatically generated for your project.  The files picker will open and list any image files you already have in your Presentation account.  You also have the option to add a new image file to use in your project.  Select an image and it will be shown in the project.  You can use the 'Eye' icon on the left to change the project image anytime you feel like it.

 

 

Next, click on the second icon, the 'File' icon.  This Files area allows you to upload files for the project.  They can be document files, spreadsheets, images, Powerpoints, videos, audio files -- any type of file you want to upload.  Files can be dragged directly from your computer and dropped into the project or you can click on the 'Files' button to open the file picker.  In the file picker, you can upload new files or select from files that are already in your Presentation account.

 

 

Once you have files in the Files area, they will be displayed in the files scroll.  You can view the files and images you have added to the project.  The '+' at the front of the scroll opens the file picker.  A preview of the currently active file is shown below the scroll.  To the right of the preview, you have the option to change the file name and write a description of the file.

 

 

When you are finished with the project, click the checkmark at the bottom to save your work and close the project.  If you just want to close it, without saving any changes, use the 'x' in the upper right corner.

 

More To Come

There are a few more icons on the left side that aren't quite ready.  Soon, you'll be able to write a reflection about the project, align it to the appropriate competencies, tag projects, categorize them, and associate any demonstrated skills.  You'll eventually be able to let other Presentation users collaborate with you on the project.

 

 

 

 

Viewing the Finished Project

After you close the project editor, you are returned to the portfolio editor and your new project is present on the page.  You can remove the project from the page using the 'x' in the upper right corner.  If you want to work on the project some more, hover over the project and click on the 'Edit Project' circle that appears.

 

 

To view the final product, use the 'preview' dropdown at the top of the editor.  Select 'view draft' and a new tab will open with the finished portfolio.

 

 

When the preview appears, click on the project to view it in the same way those you share the portfolio with will see it.  When finished, close the preview and you'll be taken back to the editor to continue working.
