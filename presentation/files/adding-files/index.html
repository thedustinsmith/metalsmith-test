Adding Files to an Electronic Portfolio - Foliotek Presentation

The Files area of your Foliotek Presentation account acts as a repository for any files you wish to upload.  It can store documents, images, videos, and links to websites.  There are two ways you can add files:  by emailing them to your account or directly uploading them into your account.

 

Adding Files via Email

You may add files to your file storage area by emailing them to add@files.foliotek.com.  To do this, open an email, attach which ever files you would like to add to your storage and send it to add@files.foliotek.com.  You must send the email from the email address listed in your account. If you don't, the system won't add the files to your account.

 

 

Adding Files to File Storage

To add files to your Foliotek file storage area while working inside a portfolio, click on the 'Files' tab and then the orange "Add Files" button to the left side of the screen.  To do this outside of a portfolio, click the 'Files' link in the top black bar to go to the file manager.

 

 

A dialog box will now appear with options for "My Computer", "Zip File", and "Web Resource" on the left side.  To add files that are on your computer, choose "My Computer", click "Choose files", use the browser to select the file(s) you wish to add.

 

 

You may now click the "Add File(s)" button and the system will put the files in your storage area.

 

 

Adding Zip Files

If you have a compressed zip folder of files you would like to add, you may add the entire folder at once by clicking "Zip File" and using the browser to find the zipped file(s).

 

 

You may now click the "Add File(s)" button just as if you were uploading a single document.

 

Adding Web Resources

To use a file that is on a different web page, you may click "Web Resource" then copy the URL into the URL box and put the name you would like the file to have in the "Friendly Name" box.  If you wish to add multiple web resources, you may click the "Add Another" link under "Friendly Name".

 

 

You may now click the "Add Web Resource(s)" button and the resource will be available in your file storage area.

 

After Your Upload

Your file storage area will now have "Uploads" selected.  To view all files again, instead of your recent uploads, click "All" in the tab list.

 

 

Adding Files From File Storage to a Presentation

To add a file in your file storage area to your presentation portfolio you need to first find the file that you want.  You can do this by browsing your files or searching using the search bar located to the top right of the files section.

 

 

Now that you have found your file, click the title to select it and your mouse will be tailed by the file name.

 

 

Click where you would like to have the file located and a link will appear that connects to that document, image, video, or website.
