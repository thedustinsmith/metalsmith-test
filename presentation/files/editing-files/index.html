Editing Files - Foliotek Presentation

To start using files in your Foliotek Presentation, first open a portofolio, then click the "FILES" button on the editing bar.

 

What Does It Mean To Edit Files?

In Foliotek, uploading your document does not have to be the end of editing that document.  If you want to change the contents of the document and re-upload the document, you can do that.  You can also rename the document directly from the file storage area.

 

Replacing Files

If you have a file that you previously uploaded, but have made changes to on your computer, you can replace the previously uploaded one, with the new updated file.  To do this, go to the file storage area, mouse over the file you wish to replace, and click "edit".

 

 

After clicking "edit", click "REPLACE FILE" under the file name.  A window will pop up allowing you to choose the file you wish to upload in its place.

 

 

Click OK and the file will be replaced.

 

Renaming A File

If you wish to rename a file you have uploaded to your portfolio, you may do this from the file storage area.  To do this, go to the file storage area, mouse over the file you wish to rename, and click "edit".

 

 

You may now type a new name for your file in the text box; click "DONE" when completed.
